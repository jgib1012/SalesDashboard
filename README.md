#  Sales Performance Dashboard (Power BI)

This interactive dashboard analyzes sales and profitability using the Sample Superstore dataset. Built in Power BI, it showcases key metrics, trends, and filters for dynamic exploration.

##  Dashboard Preview
![Sales Performance Dashboard](images/dashboard-overview.png)

##  Features
-  KPI Cards: Profit Margin, Total Sales, Total Profit, Quantity
-  Monthly Sales Trend (line chart)
-  Profit vs Sales Comparison (bar chart)
-  Region and Time Slicers (Year, Quarter)
-  Interactive filtering and drill-downs

##  Tools Used
- Power BI
- DAX (custom measure: Profit Margin)
- Sample Superstore dataset

##  Files
- `SalesDashboard.pbix`: Power BI dashboard file
- `/images/dashboard-overview.png`: Screenshot of the dashboard

##  Purpose
Designed to demonstrate end-to-end data storytelling and dashboard design for recruiters and hiring managers. Highlights ability to clean data, create custom measures, and build clear, actionable visuals.
